# Bot Aesthetics Skill

## When To Use

- Designing a Discord bot's visual identity.
- Improving embeds, rank cards, profile cards, level-up messages, welcome flows, and achievement messages.
- Creating consistent response personality and theme rules.

## Bot Identity Framework

Define:

- Primary color for neutral/info responses.
- Accent color for branded or premium moments.
- Success color for positive outcomes.
- Danger color for errors and moderation actions.
- Background color for generated cards or image assets.

Choose one emoji style and keep it consistent:

- Unicode.
- Custom server emojis.
- Flat icon style.
- Minimal no-emoji style.

## Response Personality

Pick one voice and apply it consistently:

- Friendly and casual.
- Professional and concise.
- Playful.
- RPG/fantasy.
- Premium/minimal.

## Rank And Profile Cards

- Keep avatar, username, rank/level, and progress visually separated.
- Use the bot accent color for progress bars.
- Use readable contrast on dark backgrounds.
- Export generated cards as PNG buffers.
- Avoid placing important text too close to edges.

## Level-Up And Achievement Embeds

- Use high-energy color sparingly for milestone moments.
- Mention the user clearly.
- Include the new level, reward, or achievement.
- Add timestamp and consistent footer when appropriate.

## Seasonal Themes

- Keep seasonal palettes optional and predictable.
- Do not override critical colors like danger/error.
- Use seasonal accents for celebration, not operational status.
