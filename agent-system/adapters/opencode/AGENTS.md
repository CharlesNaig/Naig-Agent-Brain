# OpenCode Adapter

Read root `/AGENTS.md` before starting an OpenCode coding session.

Session protocol:

1. Read the neutral base files in `/agent-system/base/`.
2. Read relevant skills from `/agent-system/skills/`.
3. Check `CONTEXT.md` before editing.
4. Make focused repository edits.
5. Preserve existing style and avoid unrelated rewrites.
6. After file changes, update local context and Obsidian Brain records.

Required final outputs after edits:

- `CONTEXT.md`
- `.context/YYYY-MM-DD-HHMM-short-session-title.md`
- Obsidian Brain change entry
- Obsidian `Project Index.md`
